Standard Hint

Anywhere

Displays a plain number (e.g., 1). Means exactly 1 mine exists in the 6 adjacent cells, in any configuration.

# Gurobi Base Constraint
scope_vars = [x[c] for c in hint.scope]
model.addConstr(gp.quicksum(scope_vars) == 1)

Contiguous Hint

Together { }

Displays brackets (e.g., {2}). The mines must form a single unbroken chain. We encode this by counting transitions (0 to 1) in the circular scope.

# Exactly 1 block = 2 transitions
W = [abs(y[i] - y[i+1]) for i in ring]
model.addConstr(gp.quicksum(W) == 2)

Separated Hint

Separated - -

Displays dashes (e.g., -2-). The mines cannot all be adjacent. This forces multiple blocks, meaning 4 or more transitions in the circular scope.

# >1 block = 4 or more transitions
W = [abs(y[i] - y[i+1]) for i in ring]
model.addConstr(gp.quicksum(W) >= 4)

Column Constraint

Line { }

Constrains an entire line of cells across the board. Uses the same transition logic for contiguous/separated, but pads the sequence with 0s at both ends.

# Pad sequence with fixed 0 boundaries
Y = [0] + scope_vars + [0]
W = [abs(Y[i] - Y[i+1]) for i in len(Y)-1]
model.addConstr(gp.quicksum(W) == 2)
